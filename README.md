## Projects

### 🔧 MangaMotive
**AI-powered publishing platform for anime, manga & light novel communities**

Built from scratch as an experiment in autonomous content operations. The goal: a platform that can ingest, generate, and publish structured content with near-zero manual intervention.

- **Stack:** Next.js, React, Tailwind CSS, Contentful (headless CMS), Node.js, Ollama
- **What's interesting:** Custom Node.js ingestion toolkit that processes raw text, metadata, and AI-generated outputs → fully linked Contentful entries with automated publishing, binary asset uploads, duplicate detection, and local caching
- **AI layer:** Local Ollama models generate reviews, episode summaries, and editorial articles with strict schema validation before publish — no hallucinations reaching production
- **Architecture:** Static generation + ISR + dynamic routing for SEO scalability across thousands of content pages
